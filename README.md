# pulse

Live coding music experiments with [Strudel](https://strudel.cc).

## What is Strudel?

Strudel is a JavaScript port of TidalCycles - a live coding environment for creating music through code. It runs entirely in the browser.

## Usage

1. Go to [strudel.cc](https://strudel.cc)
2. Copy/paste any `.strudel` file contents into the editor
3. Press `Ctrl+Enter` to play

## Patterns

- `patterns/` - Strudel pattern files
